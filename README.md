# EasyTime - Sistema de Gestão de Barbearia

Bem-vindo ao EasyTime, um projeto desenvolvido em Django e Bootstrap para facilitar o gerenciamento de agendamentos de uma barbearia. Este README fornece informações essenciais sobre como configurar e executar o projeto em seu ambiente local.

## Como Rodar o projeto

O projeto já conta com um banco de dados pré-configurado. Para inicializá-lo, utilize o comando:
py manage.py runserver

Após isso, o sistema estará pronto para ser usado no seu servidor local: `http://127.0.0.1:8000/`
